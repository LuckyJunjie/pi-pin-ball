## 📋 测试用例列表

### P0 - 核心功能 (必须测试)

| # | 用例 | 场景描述 | 预期结果 |
|---|------|----------|----------|
| P0-01 | 主菜单 | 打开游戏，显示主菜单 | 显示: 开始游戏、设置、排行榜 |
| P0-02 | 开始游戏 | 点击开始游戏 | 显示游戏界面、球、挡板 |
| P0-03 | 发射球 | 按住发射键蓄力，松开发射 | 球被发射向挡板区域 |
| P0-04 | 挡板控制 | 按左/右键控制左右挡板 | 挡板响应按键动作 |
| P0-05 | 得分 | 球碰撞得分区域 | 分数增加，显示得分动画 |
| P0-06 | 游戏结束 | 球掉落到底部 | 显示游戏结束界面 |

### P1 - 重要功能

| # | 用例 | 场景描述 | 预期结果 |
|---|------|----------|----------|
| P1-01 | 角色选择 | 主菜单点击角色选择 | 显示角色列表 (5个角色) |
| P1-02 | 选择角色 | 选择"幸运儿"角色 | 角色被选中，倍率显示1.2x |
| P1-03 | 角色特效 | 使用"磁铁人"角色 | 球被吸附到挡板 |
| P1-04 | 关卡选择 | 主菜单点击关卡 | 显示关卡列表 (5关) |
| P1-05 | 切换关卡 | 选择第2关 | 切换到第2关主题 |
| P1-06 | 设置菜单 | 主菜单点击设置 | 显示音量、画质等选项 |
| P1-07 | 暂停 | 游戏按暂停键 | 显示暂停菜单 |
| P1-08 | 继续游戏 | 暂停时点击继续 | 返回游戏 |

### P2 - 扩展功能

| # | 用例 | 场景描述 | 预期结果 |
|---|------|----------|----------|
| P2-01 | 排行榜 | 主菜单点击排行榜 | 显示最高分列表 |
| P2-02 | 每日挑战 | 每日挑战入口 | 显示今日挑战任务 |
| P2-03 | 成就系统 | 成就入口 | 显示已获得/未获得成就 |
| P2-04 | 统计信息 | 统计入口 | 显示游戏时长、得分等 |
| P2-05 | 音效测试 | 开启音效游玩 | 听到碰撞/得分音效 |
| P2-06 | BGM测试 | 开启音乐游玩 | 听到背景音乐 |
